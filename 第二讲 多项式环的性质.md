- _Motivation：把 $\mathbb{Z}$ 上的带余除法搬过来。_

#### 定义 & 定理：多项式的带余除法

内容：设 $f, g \in F[x], g \neq 0$，则 $\exists ! q, r \in F[x], \text{s.t. } f = gq + r, \deg r < \deg g$。其中 $q, r$ 分别称作 $f$ 除以 $g$ 的商和余式。

证明：

- 唯一性：假设 $f = gq_1 + r_1 = gq_2 + r_2$ 且 $(q_1, r_1), (q_2, r_2)$ 均满足上述条件。
- 则 $g(q_1 - q_2) = r_2 - r_1$，考察等号两边的次数即得 $q_1 = q_2, r_1 = r_2$。
- 存在性：
- (1) 当 $\deg f < \deg g$，取 $q = 0, r = f$ 即可。
- (2) 当 $\deg f \geq \deg g$，设 $f = \displaystyle\sum_{i = 0}^n a_i x^i, g = \displaystyle\sum_{i = 0}^m b_i x^i \ (n \geq m; a_n, b_m \neq 0)$。
- 考虑对 $\deg f = n$ 归纳：考察 $f - \dfrac{a_n}{b_m} g x^{n - m}$ 除以 $g$ 的商 $q_1$ 和余式 $r_1$，则 $f = g(q_1 + \dfrac{a_n}{b_m} x^{n - m}) + r_1$。故取 $q = q_1 + \dfrac{a_n}{b_m} x^{n - m}, r = r_1$ 即可。
- 综合 (1)(2) 可知这样的 $(q, r)$ 存在。

由于不保证非零元皆可逆，在 $R[x]$ 上未必能做带余除法。

#### 定义：整除

称 $g$ 整除 $f$（记作 $g \mid f$），若 $\exists q \in F[x], \text{s.t. } f = gq$。

由定义容易推知以下基本性质：

- (1) $0 \mid f \Leftrightarrow f = 0$。
- (2) 当 $g \neq 0$，$g \mid f \Leftrightarrow g$ 除 $f$ 的余式为 $0$。
- (3) 零次多项式整除任一多项式。
- (4) 整除关系有传递性，即：$f \mid g, g \mid h \Rightarrow f \mid h$。
- (5) 若 $f = \displaystyle\sum_{i = 1}^n g_i$，且 $\varphi \mid f, g_1, \dots, g_{n-1}$，则 $\varphi \mid g_n$。

#### 定义：相伴

称 $f$ 与 $g$ 相伴（记作 $f \sim g$），若 $f \mid g$ 且 $g \mid f$。

由定义容易推知以下基本性质：

- (1) $f \sim g \Leftrightarrow \exists c \in F^*, \text{s.t. } f = cg$。
- (2) 任一多项式都与一个首一（即首项系数为 $1$）多项式相伴。特别地，这里认为 $0$ 为首一多项式。

#### 定义：多项式环上的同余关系

对于 $f \in F[x] \backslash \{0\}$，定义二元关系 $\equiv_f$：

称 $g_1 \equiv_f g_2$，若 $f \mid (g_1 - g_2) \Leftrightarrow f$ 除 $g_1, g_2$ 的余式相同。

$\forall g \in F[x]$，令 $r$ 为 $f$ 除 $g$ 的余式，记 $g$ 所在的关于 $\equiv_f$ 的等价类为 $\bar{g}$，则有：

$$
\bar{g} = \{h \mid h \sim g\} = \{fq + r \mid q \in F[x]\} = \bar{r}
$$

记 $F[x] / f(x) F[x]$ 为所有上述等价类构成的集合，随后定义上的 $+, \cdot$：

$$
\left\{
\begin{array}{l}
\overline{g_1} + \overline{g_2} = \overline{g_1 + g_2} \\
\overline{g_1} \cdot \overline{g_2} = \overline{g_1 \cdot g_2}
\end{array}
\right.
$$

容易验证 $(F[x] / f(x) F[x], +, \cdot)$ 为**交换幺环**。

#### 定义：公因式与最大公因式

称 $d \in F[x]$ 为 $f, g \in F[x]$ 的公因式，若 $d \mid f$ 且 $d \mid g$。

称 $d$ 为 $f, g$ 的最大公因式，若 $d$ 为 $f, g$ 的公因式，且 $\forall f, g$ 的公因式 $\varphi$，都有 $\varphi \mid d$。

- 这里的“最大”可以理解为在相伴意义下，偏序关系 $\mid$ 上 $f, g$ 的公因式集合中的最大元。
- 下面我们将证明这个最大元存在且唯一。

#### 定理：最大公因式的唯一性

内容：若最大公因式存在，则其在相伴意义下唯一。

证明：若 $d_1, d_2$ 均为 $f, g$ 的最大公因式，则 $d_1 \mid d_2$ 且 $d_2 \mid d_1$，即 $d_1 \sim d_2$。

------

因此，当 $f, g$ 的最大公因式存在，记 $(f, g)$ 表示其中的首一多项式。

#### 引理：取余不改变最大公因式

内容：若 $f = qg + r$ 且 $(g, r)$ 存在，则 $(f, g) = (g, r)$。

#### Bézout 定理

内容：$\forall f, g \in F[x]$，$(f, g)$ 存在，且有：

$$
\exists u, v \in F[x], \text{s.t. } uf + vg = (f, g)
$$

证明：

- (1) 若 $f = 0$ 或 $g = 0$（不妨假设前者成立)，则 $(f, g) = g = 0 \cdot f + 1 \cdot g$。
- (2) 若 $f, g \neq 0$，不妨设 $\deg f \geq \deg g$。
- 设 $f = qg + r, \deg r < \deg g$。
- 考虑对 $\deg g$ 归纳：由归纳假设可知 $(g, r)$ 存在，设 $ug + vr = (g, r)$。
- 则由引理可知 $(f, g) = (g, r)$ 且 $ug + v(f - gq) = (f, g) \Leftrightarrow vf + (u - vq)g = (f, g)$，遂得证。

#### 定义：互素

称 $f, g$ 互素，若 $(f, g) = 1$。

由 Bézout 定理可推知：

$$
(f, g) = 1 \Leftrightarrow \exists u, v \in F[x], \text{s.t. } uf + vg = 1
$$

#### 命题：互素与整除的基本性质

内容：

- (1) 若 $(f, g) = 1$，则 $f \mid gh \Rightarrow f \mid h$。
- (2) 若 $f \mid h, g \mid h$ 且 $(f, g) = 1$，则 $fg \mid h$。

证明：

- (1) 由 $(f, g) = 1$ 可知：

$$
\exists u, v \in F[x], \text{s.t. } uf + vg = 1
$$

- 故有 $h = (uf + vg)h = uh \cdot f + v \cdot gh$。
- 又由于 $f \mid gh$，可知 $f \mid h$。
- (2) 设 $h = fp$。
- 因为 $(f, g) = 1, g \mid h = fp$，由 (1) 可知 $g \mid p$。
- 设 $p = gq$，则有 $fg \mid fgq = fp = h$。

------

公因式，最大公因式，互素等概念可推广到 $n \geq 3$ 个多项式的情况：

$$
(x_1, x_2, \cdots, x_n) = (x_1, (x_2, \cdots, x_n \cdots))
$$

同时，可将 Bézout 定理推广为:

$$
\exists u_1, \dots, u_n \in F[x], \text{s.t. } \sum_{i = 1}^n u_i x_i = (x_1, \cdots, x_n)
$$

- _Motivation：把 $\mathbb{Z}$ 上的素数搬过来。_

#### 定义：平凡因子

$\forall f \in F[x]$，可知 $\forall c \in F^*$，$c, cf$ 均为 $f$ 的因子，称为 $f$ 的平凡因子。

#### 定义：多项式的可约性

称 $f \in F[x]$ 为不可约多项式，若 $f$ 只有平凡因子；否则称 $f$ 为可约多项式。

#### 定理：不可约的等价条件

内容：下列四个命题等价:

- (1) $p$ 不可约。
- (2) $\forall f \in F[x], (p \mid f) \lor ((p, f) = 1)$。
- (3) $p \mid fg \Rightarrow (p \mid f) \lor (p \mid g)$。
- (4) $p$ 不可分解为更低次的非常数多项式之积。

证明：

- $(1) \Rightarrow (2)$：由于 $(p, f) \mid p$，有 $(p, f) = 1$ 或 $(p, f) \sim p$。
- 后者蕴涵 $p \mid f$，遂得证。
- $(2) \Rightarrow (3)$：若 $p \nmid f$，则 $(p, f) = 1$.
- 又由于 $p \mid fg$，有 $p \mid g$。
- $(3) \Rightarrow (4)$：假设 $p = p_1 p_2 \ (\deg p_1, \deg p_2 > 0)$。
- 则 $p \mid p_1 p_2 \Rightarrow p \mid p_1$ 或 $p \mid p_2$，但 $\deg p = \deg p_1 + \deg p_2 > \deg p_1, \deg p_2$，矛盾！
- $(4) \Rightarrow (1)$：假设 $p$ 可约，则 $\exists p_1 \mid p, \text{s.t. } 0 < \deg p_1 < \deg p$。
- 设 $p = p_1 p_2$，则 $0 < \deg p_2 < \deg p$，矛盾！

**注意：可约性依赖于域 $F$！**

- 例如：$x^2 + 1$ 在 $\mathbb{Q}[x]$ 中不可约，但在 $\mathbb{C}[x]$ 中可约。

#### 多项式环的唯一分解定理

内容：$\forall f \in F[x] \backslash \{0\}$，$f$ 存在相伴意义下唯一的不可约分解，即：

- (1) 存在性：$\exists \{p_i\}_{i = 1}^n \subset F[x], \text{s.t. } f = \displaystyle\prod_{i = 1}^n p_i$，其中 $p_i$ 不可约。
- (2) 唯一性：若 $\{q_i\}_{i=1}^m$ 亦然，则 $n=m$，且可通过调整 $\{q_i\}$ 的次序，使得 $p_i \sim q_i, \forall 1 \leq i \leq n$。

证明

- (1) 存在性：考虑对 $\deg f$ 归纳。
- I. 若 $f$ 不可约, 则 $f=f$ 即为一分解。
- II. 若 $f$ 可约，令 $f = f_1 f_2 \ (\deg f_1, \deg f_2 > 0)$。
- 由归纳假设，$f_1, f_2$ 也有不可约分解，取 $f_1, f_2$ 的任一分解式之积即得 $f$ 的一个不可约分解。
- (2) 唯一性：设 $f = \displaystyle\prod_{i = 1}^n p_i = \prod_{i = 1}^m q_i$，考虑对 $n$ 归纳。
- I. 若 $n = 0$：当 $m=0$，结论成立；否则，矛盾！
- II. 若 $n > 0$：当 $m = 0$，矛盾！
- 否则，由 $p_1$ 不可约且 $p_1 \mid f = \displaystyle\prod_{i = 1}^m q_i$，可知 $\exists 1 \leq i \leq m, \text{s.t. } p_1 \mid q_i$。不妨设 $i = 1$。
- 由于 $p_1, q_1$ 均不可约且 $p_1 \mid q_i$，有 $p_1 \sim q_1$。
- 对 $\displaystyle\prod_{i = 2}^n p_i = \prod_{i = 2}^m q_i$ 应用归纳假设即可。

#### 定义：多项式的标准不可约分解式

$\forall f \in F[x] \backslash \{0\}$，称 $f = c \displaystyle\prod_{i = 1}^n p_i^{e_i}$ 为其标准不可约分解式，若 $c \in F^*$，$p_i$ 为两两不同的不可约多项式，且 $e_i \geq 1$。

任取 $f$ 的不可约分解式，将各不可约因子首一化并提取出 $c$，再合并相同的因子即可得到其标准不可约分解式。

[例 1] 求证：对于 $f \in F[x] \backslash \{0\}$，$F[x] / f(x) F[x]$ 为域 $\Leftrightarrow f$ 不可约。

证明：

- $\Rightarrow$：假设 $f = f_1 f_2 \ (\deg f_1, \deg f_2 > 0)$，则可同 $\mathbb{Z} / n \mathbb{Z}$ 上的做法由 $\overline{f_1} \overline{f_2} = \overline{0}$ 推出矛盾。
- $\Leftarrow$：$\forall \bar{g} \in (F[x] / f(x) F[x]) \backslash \{\bar{0}\}$，有 $f \not\mid g$，则由 Bézout 定理可知 $\exists u, v \in F[x], \text{s.t. } uf + vg = 1$。
- 对等式两边同时取等价类，得到 $\bar{g}^{-1} = \bar{v}$，故 $F[x] / f(x) F[x]$ 为域。

------

- _Motivation：标准不可约分解式中出现多次的因子可能会引入某种“奇异性”，如特征多项式有重因子的矩阵不可相似对角化。_

#### 定义：$k$ 重因子

设不可约多项式 $p$ 为多项式 $f$ 的 $k$ 重因子，若 $p^k \mid f$ 且 $p^{k + 1} \not\mid f$。特别地：

- 当 $k=0$，称 $p$ 与 $f$ 互素。
- 当 $k=1$，称 $p$ 为 $f$ 的单因子。
- 当 $k \geq 2$，称 $p$ 为 $f$ 的重因子。
