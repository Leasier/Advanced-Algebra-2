- _Motivation：从 $\mathbb{Z}$ 上把一些东西搬过来（确信）_

#### 定义 & 定理：多项式的带余除法

内容：设 $f, g \in F[x], g \neq 0$，则 $\exists! q, r \in F[x], \text{s.t. } f = gq + r, \deg r < \deg g$。其中 $q, r$ 分别称作 $f$ 除以 $g$ 的商和余式。

证明：

- 唯一性：假设 $f = gq_1 + r_1 = gq_2 + r_2$ 且 $(q_1, r_1), (q_2, r_2)$ 均满足上述条件。
- 则 $g(q_1 - q_2) = r_2 - r_1$，考察等号两边的次数即得 $q_1 = q_2, r_1 = r_2$。
- 存在性：
- (1) 当 $\deg f < \deg g$，取 $q = 0, r = f$ 即可。
- (2) 当 $\deg f \geq \deg g$，设 $f = \displaystyle\sum_{i = 0}^n a_i x^i, g = \displaystyle\sum_{i = 0}^m b_i x^i \ (n \geq m; a_n, b_m \neq 0)$。
- 考虑对 $\deg f = n$ 归纳：考察 $f - \dfrac{a_n}{b_m} g x^{n - m}$ 除以 $g$ 的商 $q_1$ 和余式 $r_1$，则 $f = g(q_1 + \dfrac{a_n}{b_m} x^{n - m}) + r_1$。故取 $q = q_1 + \dfrac{a_n}{b_m} x^{n - m}, r = r_1$ 即可。
- 综合 (1)(2) 可知这样的 $(q, r)$ 存在。

由于不保证非零元皆可逆，在 $R[x]$ 上未必能做带余除法。
